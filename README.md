# Blassti AI

Blassti AI is an upgraded version of Blassti, an event management web platform built with Vue.js and Vuetify.

The platform allows users to discover events, book tickets, reserve venues, create bundles, join carpools, receive notifications, and interact with different roles such as visitors, event goers, organizers, venue sellers, and administrators.

This version adds an AI-powered bundle generator. Users can describe their preferences such as event type, budget, location, date, and mood, and the system generates a personalized event bundle using a Python backend connected to the OpenAI API.

## Main Features

- Event discovery and browsing
- Ticket booking
- Seat selection
- Venue reservation
- Event bundles
- AI-powered bundle generation
- Carpool system
- Notifications
- Reports and moderation
- Admin statistics
- Mobile and tablet responsive design
